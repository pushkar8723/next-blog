# Building Accessible React Components

Web accessibility ensures that everyone, including people with disabilities, can use your website. Let's explore how to build accessible React components.

## Why Accessibility Matters

Accessibility benefits everyone:

- Users with visual impairments using screen readers
- Users with motor disabilities using keyboard navigation
- Users with cognitive disabilities
- Temporary disabilities (broken arm, lost glasses)
- Situational limitations (bright sunlight, noisy environment)

## Semantic HTML

Start with proper semantic HTML:

```tsx
// Good
<button onClick={handleClick}>Submit</button>

// Bad - div doesn't have button semantics
<div onClick={handleClick}>Submit</div>
```

## Keyboard Navigation

Ensure all interactive elements are keyboard accessible:

```tsx
function Menu() {
  const [activeIndex, setActiveIndex] = useState(0)

  const handleKeyDown = (e: React.KeyboardEvent) => {
    switch (e.key) {
      case "ArrowDown":
        setActiveIndex((prev) => Math.min(prev + 1, items.length - 1))
        break
      case "ArrowUp":
        setActiveIndex((prev) => Math.max(prev - 1, 0))
        break
      case "Enter":
        selectItem(activeIndex)
        break
    }
  }

  return (
    <ul role="menu" onKeyDown={handleKeyDown}>
      {items.map((item, index) => (
        <li
          key={item.id}
          role="menuitem"
          tabIndex={index === activeIndex ? 0 : -1}
        >
          {item.label}
        </li>
      ))}
    </ul>
  )
}
```

## ARIA Attributes

Use ARIA attributes when semantic HTML isn't enough:

```tsx
function Modal({ isOpen, onClose, title, children }) {
  return (
    <div
      role="dialog"
      aria-modal="true"
      aria-labelledby="modal-title"
      aria-hidden={!isOpen}
    >
      <h2 id="modal-title">{title}</h2>
      {children}
      <button onClick={onClose} aria-label="Close modal">

    </div>
  )
}
```

## Focus Management

Manage focus appropriately:

```tsx
function Dialog({ isOpen, onClose }) {
  const closeButtonRef = useRef<HTMLButtonElement>(null)

  useEffect(() => {
    if (isOpen) {
      closeButtonRef.current?.focus()
    }
  }, [isOpen])

  return (
    <div role="dialog">
      <button ref={closeButtonRef} onClick={onClose}>
        Close
      </button>
    </div>
  )
}
```

## Conclusion

Building accessible components requires intention and practice. Start with semantic HTML, ensure keyboard navigation works, and use ARIA attributes wisely. Your users will thank you!
